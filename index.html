발주서 Builder 사용법 안내
모든 값은 발주서에서 직접 수정하여 출력할 수 있습니다.
단, 수정 사항은 에어테이블(발주리스트)에는 반영되지 않습니다.

반드시 입력된 값이 정확한지 확인 후 출력하여 발송해주세요.

색상의 의미

노란색 — 에어테이블에서 가져온 값
파란색 — ‘배송지 선택’에서 설정한 값

노란색 칸에는 공급사명 · 담당/연락처 · 발주일 · 발주(시안) 담당자 · 발주 품목/상세스펙/수량이 에어테이블에서 자동으로 채워집니다.

사용 순서

  1. 노란색·파란색 칸의 값을 확인하고, 필요하면 칸을 클릭해 수정합니다.
  2. ‘배송지 선택’에서 배송지를 고르면 주소·연락처가 채워지고, ‘배송 담당자’는 목록에서 선택(또는 직접 입력)합니다.
  3. 특이사항은 체크박스로 표시 여부를 정하고, 행 추가·삭제가 가능합니다.
  4. ‘미리보기’로 출력 모습을 확인한 뒤 ‘인쇄 / PDF 저장’ 합니다.

도움 기능

  • 잘못 고쳤다면 ↶ 되돌리기(Ctrl+Z) / ↷ 다시 실행.
  • 처음 상태로 돌리려면 ‘기본값 복원’.
